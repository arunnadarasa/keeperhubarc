# Para Integration (Discontinued)

> **Para has been discontinued.** KeeperHub no longer supports Para as a wallet provider. [Turnkey](/wallet-management/turnkey) is the default wallet provider for all new and existing accounts.

If you previously used a Para wallet with KeeperHub, please contact support to migrate to Turnkey.
